# ecole/

Livrables pour l'Apreneur Académie.

Ce qui va ici :
- Programmes et plans de formation
- Modules de cours (textes, exercices, quizz)
- Emails de séquence ou nurturing
- Pages de vente pour les formations
- Supports pédagogiques (fiches, templates étudiants)
- Contenus communautaires (posts, newsletters)

Ce qui ne va pas ici : les contenus ou ressources que tu fournis → `context/import/`
