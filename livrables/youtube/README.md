# youtube/

Tout le contenu éditorial lié à la chaîne YouTube.

Ce qui va ici :
- Scripts de vidéos (intro, développement, outro)
- Hooks d'accroche (variations A/B)
- Briefs vidéos (angle, structure, ressources)
- Titres et descriptions optimisés SEO YouTube
- Calendrier éditorial
- Idées de vidéos et angles traités

Convention recommandée : préfixer par le numéro d'épisode si applicable
→ `2026-06-01_ep12_script-automatiser-notion_v1`

Ce qui ne va pas ici : les notes ou références que tu fournis → `context/import/`
